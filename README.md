# Princípios de Construção de Agentes de IA

Este repositório contém exemplos práticos e projetos relacionados aos princípios fundamentais para construir agentes de inteligência artificial.

## 📚 Projetos

### ✅ Disponíveis

- **[Agent](./agent/)** - Exemplos de agentes de IA implementados
  - **Agente de Clínica Médica**: Um assistente virtual que demonstra como criar um agente conversacional para atendimento em clínicas. O exemplo inclui identificação de pacientes (nome e CPF), agendamento de consultas e retornos, consulta de exames e resultados, visualização de agendas disponíveis dos médicos, e conversação natural usando OpenAI GPT. Ideal para entender como integrar LLMs com bancos de dados e criar fluxos de atendimento automatizados.
